# Pokémon API Flutter

Una aplicación móvil desarrollada en **Flutter** que consume la [PokéAPI](https://pokeapi.co/) para mostrar una lista de Pokémon y permite buscar Pokémon específicos por nombre. La app también incluye funcionalidad de prueba con imágenes aleatorias de perros usando [Dog API](https://dog.ceo/dog-api/).

## Funcionalidades

- Listado de los primeros 50 Pokémon con sus imágenes.
- Búsqueda de Pokémon por nombre.
- Tarjetas detalladas con información del Pokémon:
  - Nombre
  - Altura
  - Peso
  - Tipo(s)
- Compatible con Android y web.
- Diseño sencillo y limpio usando Material Design.

---

## Tecnologías

- **Flutter 3.x**
- **Dart 3.x**
- **http**: Para realizar las peticiones HTTP.
- **Material Design**: Para la UI.
- Compatible con **Android** y **Web**.
